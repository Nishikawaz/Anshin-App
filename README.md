# Anshin

App de finanzas personales para Paraguay.

Este repositorio arranca con una base Android en Kotlin + Compose y el primer núcleo de parsers bancarios para Itaú, Ueno y Continental.

## Estado actual

- Documentación base del proyecto incluida
- Estructura inicial Android/Gradle creada
- Parsers bancarios implementados con tests unitarios

- Implementar `NotificationListenerService`
- Conectar dashboard y flujo de confirmación de transacciones
